# Fork Me

Ambiguity is a fork. Expose it.

1. Burn down discoverable facts and reversible conventions.
2. State one unresolved choice and what it blocks.
3. Present at least three materially different options.
4. Put the recommendation first.
5. Explain each option's reason and cost in one plain sentence.
6. Use the harness-native picker and its `Other` option.
7. Batch independent forks. Resolve dependent forks in order.
8. Freeze dependent work until selection.

If the picker is unavailable, report the capability blocker. Never guess, infer preference, or ask
for an open-text answer.
